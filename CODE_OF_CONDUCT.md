# Code of Conduct

## 1. Purpose
This project is committed to providing a welcoming, safe, and professional environment for everyone who participates—regardless of background, identity, or level of experience.

We value **constructive collaboration**, **clear technical debate**, and **respectful communication**. We do not tolerate harassment or abuse.

## 2. Our Standards
Examples of behavior that contributes to a positive environment include:

- Being respectful, considerate, and kind
- Assuming good intent and asking clarifying questions
- Focusing on facts, reproducible evidence, and technical substance
- Giving and receiving feedback professionally
- Respecting boundaries, privacy, and confidentiality when applicable
- Keeping discussions on-topic and helpful

Examples of unacceptable behavior include:

- Harassment, discrimination, or hate speech
- Personal attacks, insults, humiliation, or threats
- Trolling, persistent bad-faith argumentation, or intentional disruption
- Publishing private information (doxxing) or confidential material without permission
- Sexualized language or unwelcome sexual attention
- Encouraging or enabling illegal activity

## 3. Scope
This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub issues, pull requests, code reviews, discussions, and comments
- Project documentation and community channels
- Any official project communication where participants represent the project

## 4. Enforcement
Project maintainers are responsible for clarifying standards and enforcing this Code of Conduct.

If you experience or witness unacceptable behavior, please report it as soon as possible.

Reports will be handled with discretion. We will consider privacy and safety, and we will act in a timely manner.

## 5. Enforcement Guidelines
Maintainers may take any action they deem appropriate, including:

### Level 1 — Correction
**Impact:** Minor, unintentional violation  
**Consequence:** Private or public reminder of expected behavior

### Level 2 — Warning
**Impact:** Repeated or more serious violation  
**Consequence:** Formal warning; conditions for continued participation

### Level 3 — Temporary Restrictions
**Impact:** Sustained disruption, harassment, or hostile behavior  
**Consequence:** Temporary limitation of participation (e.g., limited commenting, temporary ban)

### Level 4 — Permanent Ban
**Impact:** Severe harassment, threats, or sustained harmful conduct  
**Consequence:** Permanent removal from project spaces

## 6. Maintainer Discretion
Maintainers may:
- remove, edit, or reject contributions that violate this Code of Conduct
- lock conversations, close issues, or revert changes when necessary
- take action that protects the project and its participants

## 7. Attribution
This Code of Conduct is inspired by common open-source community standards and adapted for this project.
